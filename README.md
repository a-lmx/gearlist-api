# gearlist-api

###Endpoints

MVP

- list of users lists
  - get '/users/:id/lists'

- Information about a list
  - get '/lists/:id/'
  - or get 'lists/:id/items' ?

- create new list
  - post '/lists'

- add items to list
  - post '/list_items'

- create new item
  - post '/items'

- remove item from list
  - delete '/list_items/:id'

ADD LATER

- delete list
  - delete '/lists/:id'

- delete item
  - delete '/items/:id'

- update item
  - put/patch '/items/:id'
